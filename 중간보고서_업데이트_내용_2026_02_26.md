## 🆕 새로 추가된 기능

### 1. BGM 관리 시스템 (신규)
**목적**: SageMaker에서 생성한 BGM을 DynamoDB에 저장

**API 엔드포인트**:
- `POST /api/v1/bgm/update` - BGM URL 업데이트
- `GET /api/v1/bgm/{user_id}` - BGM URL 조회

**저장 구조**:
- DynamoDB `user_friends` 테이블 활용
- `friend_user_id == user_id`인 모든 레코드에 BGM URL 저장
- Self-friend 레코드 + 모든 친구의 레코드에 동시 업데이트

**SageMaker 연동 플로우**:
```
1. SageMaker에서 BGM 생성
   ↓
2. S3에 BGM 파일 업로드
   ↓
3. POST /api/v1/bgm/update 호출
   ↓
4. DynamoDB user_friends 테이블 업데이트
   ↓
5. 완료
```

### 2. Audio Validation API (신규)
**목적**: Audio Guard를 독립 API로 분리

**API 엔드포인트**:
- `POST /api/v1/audio/validate` - 오디오 파일 검증

**검증 항목**:
- 길이 검증 (0.5초 ~ 5초)
- 주파수 분석 (100Hz ~ 8000Hz)
- 품질 개선 (노이즈 제거, 정규화)
- AWS Transcribe STT
- Gemini LLM 판단

**응답 형식**:
```json
{
  "is_valid": true/false,
  "validation_details": {
    "duration": 2.5,
    "frequency_range": "100-8000Hz",
    "transcription": "야옹",
    "llm_judgment": "적절한 고양이 소리"
  }
}
```

### 3. 고양이 캐릭터 생성 (Gemini 2.5 Flash)
**목적**: 사람 사진을 옷 입은 고양이 캐릭터로 변환

**API 엔드포인트**:
- `POST /api/v1/cat-character/generate/{user_id}` - 고양이 캐릭터 생성

**특징**:
- Gemini 2.5 Flash Image API 사용
- 원본 이미지는 저장하지 않음 (생성된 캐릭터만 S3 저장)
- 옷 입은 고양이 (원본 의상 반영)
- 표정, 헤어스타일, 성별 보존
- 120초 타임아웃

---

### 4. Quick Register (QR 코드 기반 등록)
**목적**: QR 코드로 빠른 회원가입 + 자동 친구 추가

**API 엔드포인트**:
- `POST /api/v1/quick-register/register` - 빠른 등록
- `GET /api/v1/quick-register/qr/{user_id}` - QR 정보 조회

**처리 흐름**:
```
1. QR 코드 스캔
   ↓
2. 닉네임 + 음성 파일 업로드
   ↓
3. Audio Guard 검증
   ↓
4. 자동 회원가입
   ↓
5. 자동 친구 추가 (고정 사용자 ID)
   ↓
6. 완료
```
